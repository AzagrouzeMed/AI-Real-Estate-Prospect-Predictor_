# AI Real Estate Prospect Predictor

This project is an Artificial Intelligence system that predicts whether a real estate prospect is likely to buy a property.

## Project Overview

The application uses Machine Learning to analyze prospect data such as age, income, marital status, number of children, budget, city, and property type.

Based on this information, the model predicts whether the client is likely to buy a property.

## Technologies Used

- Python
- Pandas
- Scikit-learn
- Streamlit
- Machine Learning

## Features

- Predict if a prospect will buy a property
- Interactive web interface with Streamlit
- Automatic dataset update
- Data visualization in the app

## How to Run the Project

1. Install dependencies

```
pip install pandas scikit-learn streamlit
```

2. Run the application

```
streamlit run app.py
```

3. Open in browser

```
http://localhost:8501
```
